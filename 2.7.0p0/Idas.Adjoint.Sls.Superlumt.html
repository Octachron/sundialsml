SuperLU_MT sparse-direct linear solver module (requires SuperLU_MT).
See sundials: The SuperLUMT Solver
val solver_csc : Sls.SparseMatrix.csc Idas.Adjoint.Sls.sparse_jac_fn ->
       nnz:int ->
       nthreads:int -> [> Nvector_serial.kind ] Idas.Adjoint.serial_linear_solver
A direct linear solver on compressed-sparse-column matrices. In the call, superlumt jfn nnz, jfn specifies a callback function that computes an approximation to the Jacobian matrix and nnz is the maximum number of nonzero entries in that matrix.
