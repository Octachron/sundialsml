Module Arkode.Mass.Alternate

module Alternate: sig .. end

Alternate mass matrix solvers

type ('data, 'kind) minit = ('data, 'kind) Arkode.session -> unit

Functions that initialize mass matrix solver data, like counters and statistics.

Raising any exception in this function (including Sundials.RecoverableFailure) is treated as an unrecoverable error.

    See sundials: minit
type ('data, 'kind) msetup = ('data, 'kind) Arkode.session -> 'data Arkode.triple -> unit

Functions that prepare the mass matrix solver for subsequent calls to lsolve.

This function may raise a Sundials.RecoverableFailure exception to indicate that a recoverable error has occurred. Any other exception is treated as an unrecoverable error.

The vectors ypred, fpred, and those in tmp should not be accessed after the function returns.

    See sundials: msetup
type ('data, 'kind) msolve = ('data, 'kind) Arkode.session -> 'data -> unit

Functions that solve the linear equation $Mx = b$. $M$ is the system mass matrix. The call lsolve s b weight has as arguments:

  • s, the solver session,
  • b, the right-hand side vector, also used for returning the result, and
  • weight, a vector containing the error weights.

Raising Sundials.RecoverableFailure indicates a recoverable error. Any other exception is treated as an unrecoverable error.

The vectors b and weight should not be accessed after the function returns.

    See sundials: msolve
    See sundials: Mass matrix solver

val solver : (('data, 'kind) Arkode.session ->
        ('data, 'kind) Nvector.t -> ('data, 'kind) callbacks) ->
       ('data, 'kind) Arkode.Mass.solver

Creates a mass matrix solver from a function returning a set of callbacks. The creation function is passed a session and a vector. The latter indicates the problem size and can, for example, be cloned.
