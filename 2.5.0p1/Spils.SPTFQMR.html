val solve : 'd t ->
       x:('d, 'k) Nvector.t ->
       b:('d, 'k) Nvector.t ->
       delta:float ->
       ?sx:('d, 'k) Nvector.t ->
       ?sb:('d, 'k) Nvector.t ->
       ?psolve:'d Spils.psolve ->
       'd Spils.atimes -> Spils.preconditioning_type -> bool * float * int * int
Solves the linear system Ax = b using the SPTFQMR iterative method. The atimes function computes the matrix vector product Ax, the other arguments are described below. The function returns a tuple (solved, res_norm, nli, nps) where solved indicates whether the system converged, res_norm is the L2 norm of the scaled preconditioned residual $\lVert s_b P_1^{-1} (b - Ax) \rVert_{L2}$ , and, nli and nps count, respectively, linear iterations performed and calls to psolve. Repeated calls can be made to solve with varying input arguments, but a new session must be created if either the problem size or the maximum Krylov dimension change.
Raises
  • ConvFailure Failed to converge
  • PSolveFailure The psolve function failed
  • ATimesFailure The atimes function failed
  • PSetFailure pset failed
See sundials: SpgmrSolve
x : initial guess on entry; result on return
b : right-hand side vector
delta : tolerance of the L2 norm: res_norm <= delta
sx : optional positive scale factors for x,
sb : optional positive scale factors for b,
psolve : optionally solves the preconditioner system.
