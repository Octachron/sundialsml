Module Cvode.Densematrix


module Densematrix: sig .. end
Operations for creating and manipulating dense matrices.
type t
This type represents a DlsMat returned from a call to Cvode.Densematrix.new_dense_mat.
See sundials: Type DlsMat
See sundials: NewDenseMat

Basic access


val new_dense_mat : int * int -> t
new_dense_mat m n returns an m by n dense matrix.
See sundials: NewDenseMat 
val print_mat : t -> unit
Prints a dense matrix to stdout.
See sundials: PrintMat 
val get : t -> int * int -> float
get a (i, j) returns the value at row i and column j in a, where 0 <= i < m and 0 <= j < n.
See sundials: DENSE_ELEM 
val set : t -> int * int -> float -> unit
set a (i, j) v stores the value v at row i and column j in a, where 0 <= i < m and 0 <= j < n.
See sundials: DENSE_ELEM

Calculations


val set_to_zero : t -> unit
Fills the matrix with zeros.
See sundials: SetToZero 
val add_identity : t -> unit
Increment a square matrix by the identity matrix.
See sundials: AddIdentity 
val copy : t -> t -> unit
copy src dst copies the contents of one matrix into another.
See sundials: DenseCopy 
val scale : float -> t -> unit
scale c a multiplies each element of a by c.
See sundials: DenseScale 
val getrf : t -> int_array -> unit
getrf a p performs the LU factorization of a with partial pivoting according to p.
Raises ZeroDiagonalElement Zero found in matrix diagonal
See sundials: DenseGETRF 
val getrs : t -> int_array -> real_array -> unit
getrs a p b finds the solution of ax = b using LU factorization. a must be a square matrix.
See sundials: DenseGETRS 
val potrf : t -> unit
Performs Cholesky factorization of a real symmetric positive matrix.
See sundials: DensePOTRF 
val potrs : t -> real_array -> unit
potrs a b finds the solution of ax = b using Cholesky factorization.
See sundials: DensePOTRS 
val geqrf : t -> real_array -> real_array -> unit
geqrf a beta work performs the QR factorization of m by n matrix, with m >= n.
See sundials: DenseGEQRF 
val ormqr : a:t ->
       beta:real_array -> v:real_array -> w:real_array -> work:real_array -> unit
ormqr a beta v w work computes the product w = Qv, with Q calculated using Cvode.Densematrix.geqrf.
See sundials: DenseORMQR
a : matrix passed to Cvode.Densematrix.geqrf
beta : vector apssed to Cvode.Densematrix.geqrf
v : vector multiplier
w : result vector
work : temporary vector used in the calculation
