Module Cvode.Bandmatrix


module Bandmatrix: sig .. end
Operations for creating and manipulating banded matrices.
type t
This type represents a DlsMat returned from a call to Cvode.Bandmatrix.new_band_mat.
See sundials: Type DlsMat
See sundials: NewBandMat

Basic access


val new_band_mat : int * int * int * int -> t
new_band_mat n mu ml smu returns an n by n band matrix of upper bandwith mu and lower bandwidth ml.
  • If smu = mu, the result will not be LU factored.
  • Otherwise pass smu = min(n-1, mu + ml).

See sundials: NewBandMat 
val print_mat : t -> unit
Prints a band matrix to stdout.
See sundials: PrintMat 
val get : t -> int * int -> float
get a (i, j) returns the value at row i and column j of the n by n matrix a, where 0 <= i, j <= n - 1 and j - mu(A) <= i <= j + ml(A).
See sundials: BAND_ELEM 
val set : t -> int * int -> float -> unit
set a (i, j) v stores the value v at row i and column j of the n by n matrix a, where 0 <= i, j <= n - 1 and j - mu(A) <= i <= j + ml(A).
See sundials: BAND_ELEM

val copy : t -> t -> int -> int -> unit
copy src dst copymu copyml copies the submatrix with upper and lower bandwidths copymu and copyml of the n by n band matrix src into the n by n band matrix dst.
See sundials: BandCopy 
val scale : float -> t -> unit
scale c a multiplies each element of a by c.
See sundials: BandScale 
val gbtrf : t -> int_array -> unit
gbtrf a p performs the LU factorization of a with partial pivoting according to p.
See sundials: BandGBTRF 
val gbtrs : t -> int_array -> real_array -> unit
gbtrs a p b finds the solution of ax = b using LU factorization. a must be a square matrix.
